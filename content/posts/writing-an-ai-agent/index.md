## Introduction

I followed [this tutorial](https://ampcode.com/notes/how-to-build-an-agent) to build my own ["mini-agent"](https://github.com/stpotter16/mini-agent) that can read and edit code.

## Learning by doing 

I was shocked by little code this took. I expected there to be _much_ more magic in the doing than there really was.

I am impressed by how smooth the SDK API made it to define and use these tools.

Ultimately this was a good reminder to press deeper into things than you might be willing to at first blush. You can often make more progress than you think.
